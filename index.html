HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements nested inside other elements are called child elements
  • The body element is an example of a semantic HTML element, whos name desribes the content and role it plays on the webpage.
  • The 'br' element creates a line of empty space, line break or two blocks of content
  • The 'a' element (anchor) creates links to the same webpage or other webpages
  • The 'href' attribute points to the URL for the link
  • When using an external CSS file you need to use the 'link' element to tie the two together

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Assigning colors in CSS can be done using semantic references like 'grey' or 'blue' or by its hexadecimal code

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all changes files to the staging area
  • git commit -m "commit message here": commits changes to repo
  • git pull origin main: sync local branch with remote branch in GitHub
  • git push origin "branch name": push changes from new branch
  • Go to repo in GitHub and create pull request with title and short description
  • Merge pull request
  • git checkout main git pull origin main: update local repo with new merged-in feature.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The === operator checks to see if two values are equal and returns a Boolean result
  • Ctrl+Shift+I will open Chrome DevTools, navigate to the Console tab to see the JaveScript logic
